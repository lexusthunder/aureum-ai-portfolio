# Contributing to Aureum AI Portfolio

This portfolio is mainly a personal showcase, but if you want to suggest fixes or extensions, here's how.

## Local dev

```bash
git clone https://github.com/Aureum/aureum-ai-portfolio.git
cd aureum-ai-portfolio
```

Each project is independent — `cd` into the one you care about and follow its `PORTFOLIO_NOTE.md`.

## Code style

- Python: black + ruff, type hints encouraged
- TypeScript: prettier + eslint
- Commits: conventional commits (`feat:`, `fix:`, `docs:`, `refactor:`)

## What I'm NOT looking for in PRs

- Style-only changes
- Auto-generated dependency bumps without rationale

## What I AM looking for

- Bug fixes with a reproducible case
- Suggestions on how to deploy a project on GCP / AWS / Azure
- Pointers to relevant Google papers I should read
