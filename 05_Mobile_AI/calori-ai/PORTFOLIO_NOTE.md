# Portfolio Note — CaloriAI

## What it is

Mobile-first AI nutrition app. Built with **React Native (Expo) + TypeScript** — see `app/diary.tsx`. Users log meals and the app uses an LLM-backed route to estimate calories and macros.

## Why it matters

Demonstrates I can ship AI features inside a real mobile app, not just a web demo. Google's mobile + AI surface area (Pixel, Android, Gemini Nano) is huge — this is the closest analog in my portfolio.

## Tech stack

`React Native` · `Expo` · `TypeScript` · `LLM API for nutrition estimation`

## Status

🟡 Working diary screen + LLM-backed estimation. Roadmap: add on-device food classification with TensorFlow Lite (a true Gemini-Nano-style use case).
